    • Rules

  • Scoring is dynamic, the points of a challenge decrease per solve
  • The flag format is: AOTW{133t_speak} unless specified otherwise
  • DO NOT share flags, cheat or spoil challenges in the chat or elsewhere
  • DO NOT use any vulnerability scanners / dirbuster / bruteforce on services. It will not get you flags
  • One account per team. Each group of people participating should use exactly one account for submitting flags. Creating multiple accounts for any reason is not allowed.
  • Organizers decisions are final. There might arise situations which are not covered by these rules in which case the decision is up to the organisers and any decisions made are final.
  • Help can be found on Discord or IRC. Challenge specific questions should be directed at the author of the challenge
